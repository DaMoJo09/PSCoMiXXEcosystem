==================================================
1. PLATFORM GOALS

Build PSCoMiXX as:

==================================================

- Framework: Next.js (App Router preferred)

- Rendering:
  - ALL marketing + marketplace pages MUST be SSR or SSG
  - NO client-only rendering for public pages
- Styling:
  - PS|MMM brutalist black/white aesthetic
  - Light/Dark mode toggle
- Hosting:
  - Replit, Vercel, or equivalent with edge caching
- Analytics:
  - Deferred, non-blocking

==================================================
4. SEO + CRAWLABILITY (NON-NEGOTIABLE)
==================================================
A. SERVER-RENDERED HTML
- View-source MUST show:
  - One H1 per page
  - Real paragraph text
  - Crawlable <a href> links
- No “image-only” pages
- No empty root div waiting for JS hydration

B. INTERNAL LINKING
- Header + footer use real anchor links
- Marketplace category → product → creator interlinking
- All key pages reachable within 2 clicks from homepage

C. CANONICALIZATION
- Enforce ONE canonical domain (www OR non-www)
- 301 redirects:
  - http → https
  - non-canonical → canonical
- <link rel="canonical"> on every indexable page

D. ROBOTS + SITEMAP
- robots.txt allows crawling of all public routes
- sitemap.xml auto-generated with:
  - All indexable pages
  - Marketplace listings
  - <lastmod> timestamps

E. METADATA (EVERY PAGE)
- Unique <title> (≤60 chars)
- Meta description (150–160 chars)
- OpenGraph:
  - og:title
  - og:description
  - og:image (1200x630, absolute URL)
  - og:url
- Twitter Cards (summary_large_image)

F. STRUCTURED DATA (JSON-LD)
- Organization schema (PSCoMiXX)
- WebSite schema
- SoftwareApplication schema
- Product schema for marketplace items:
  - name
  - description
  - image
  - price
  - currency
  - availability
  - seller
- AggregateRating schema (when reviews exist)

==================================================
5. MARKETPLACE ARCHITECTURE & CONVERSION
==================================================
A. MARKETPLACE LISTING PAGES MUST INCLUDE
- Product title
- Visual preview (image/video)
- Short benefit-driven summary
- “What’s included” bullets
- Compatible formats (Comic / VN / CYOA)
- Creator profile card
- License clarity
- Price
- CTA: Buy / Add to Library / Use in Project

B. TRUST SIGNALS
- “Made for PSCoMiXX” badge
- Creator verification badge
- Usage count / installs
- Featured creators
- Refund + license policy visibility

C. FRICTION REDUCTION
- Guest browsing allowed
- Login only at checkout or save-to-library
- Checkout ≤ 2 steps
- Clear post-purchase flow:
  “Open in CoMiXX” / “Download Pack”

D. UPSELL & RETENTION
- “Used in these projects”
- “Creators also bought”
- Bundles (Starter Kits, Genre Packs)
- Featured collections
- Limited-time promos

==================================================

- CLS < 0.1
